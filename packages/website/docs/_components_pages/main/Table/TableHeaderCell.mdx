### Horizontal cell alignment

Control the horizontal alignment of cells by utilizing the `hAlign` property.

Please note that the behaviour of `hAlign` depends on where you set it:
1. `hAlign` is set on `TableHeaderCell` level<br />
Changes the horizontal alignment of the `TableHeaderCell` and their corresponding `TableCells`.
2. `hAlign` is set on `TableCell` level only<br />
The horizontal alignment is only changed for this one `TableCell`
3. `hAlign` is set on `TableHeaderCell` and `TableCell` level<br />
Changing the `hAlign` property on `TableHeaderCell` level changes only the `TableHeaderCell` itself and those `TableCells` without a `hAlign` property.
